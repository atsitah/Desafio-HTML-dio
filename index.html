O que é o HTML

inicio

O termo HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto).
Como o nome já indica, o HTML não é uma linguagem de programação mas sim uma linguagem de marcação, ou seja, ela é usada para estruturar uma página.

Sua estrutura básica é composta por tags chamadas:

  • html - que marca o único do do documento
  • head - que é o cabeçalho do documento
  • title - o título que aparecerá na aba do navegador
  • body - local onde será inserido o conteúdo

Tags

inicio

As tags são comandos que irão estruturar os conteúdos das páginas.
Tags são colocadas entre os sinais de maior que e menor que < >.
Algumas das tags vistas neste curso foram:

  • h1, h2, h3 … - títulos da página, a numeração indica a hierarquia do título
  • p - parágrafo do texto
  • a - ancoragem, cria um link para uma página externa ou para dentro do mesmo documento
  • id - cria um identificador para um item do documento que poderá ser usado para diferentes objetivos

Textos

inicio

Os textos de conteúdo devem ser adicionados dentro da tag body.
É possível formatar o texto usando tags como: Strong, para negrito; i, para itálico; u, para o sublinhado; mark, para marcar o texto; entre outras
tags de formatação que irão ajudar a dar mais estilo e trazer mais significado para o conteudo.

Das diversas tags utilziadas estão incuídas as tags de organização do texto:
h1, h2, h3, p, a, e tags de formatação como strong, i, b, u.
Incluindo também a tag blockquote que possibilita criar esta citação.

Listas ordenadas e não ordenadas

inicio

No texto também é possível criar listas, elas se dividem em dois tipos:

Listas ordenadas "ol": cria uma lista com ordem, enumerando a lista;
Listas não ordenadas "ul": cria uma lista apenas com um marcador;

Para que a lista funcione é necessário adicionar os itens em uma tag filha, "li".

Ancoragem

inicio

As tags “a” também chamada de ancora.
Esta tag tem como função criar uma ancoragem entre um link externo ou itens do mesmo documento criando assim um link que irá direcionar para o local indicado;
